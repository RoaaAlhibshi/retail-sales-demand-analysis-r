# Retail Sales Demand Analysis (R)

## Business Context
Understanding daily retail sales patterns is essential for inventory planning, staffing decisions, and demand management.  
This project analyzes real daily sales data to identify trends, seasonality, and demand behavior, and produces forecasts to support business decision-making.

## Goal
- Analyze real daily retail sales data
- Identify trends and seasonal patterns
- Forecast short-term demand
- Translate analytical results into clear business insights

## Dataset
Real-world daily retail sales data (date and daily sales), provided in Excel format.

## Project Structure
- `R/` : R scripts (step-by-step analysis)
- `data/raw/` : original sales data (Excel)
- `reports/figures/` : plots and visual outputs
- `reports/metrics/` : model evaluation metrics

## Tools
R, dplyr, lubridate, ggplot2, forecast
